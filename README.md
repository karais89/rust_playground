## Hello, Cargo

```
cargo new hello_cargo
```

- vscode에서 launch.json 설정이 조금 힘든 부분이 있다.
```
<root>
L <project1>
L <project2>
```
구조라면 root에서 launch.json을 실행하지 말고 각 프로젝트 별 launch.json을 생성하고, 각 프로젝트를 별도의 vscode로 열어서 실행하는 방식을 취해야 될 것으로 보인다.
